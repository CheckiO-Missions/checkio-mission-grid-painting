Diese Mission malt bestimmte Zellen in einem 5x5-Raster mit einem Pinsel. Und geben Sie die minimale Anzahl von Schritten zurück.

  • Jeder Zelle wird ein Buchstabe des Alphabets von A bis Y zugewiesen (in der Reihenfolge der Zeilengröße). Verwenden Sie diesen als Eingabewert.
  • Sie können vertikal (oder horizontal) aufeinanderfolgende Zellen mit einem Pinsel beliebiger Breite auf einmal malen.

HINWEIS:

  • Malen Sie keine unbenannten Zellen.
  • Die Zelle kann mehrfach gezeichnet werden.

Eingabe: Bezeichnete Zellen (str).

Ausgabe: Minimale Anzahl von Schritten (int).
