This mission paints a designated cells in a 5x5 grid with a brush. And return its minimum number of steps.

  • Each cell is assigned a letter of the alphabet from A to Y (Row-major order). Use this as input value .
  • You can paint vertically (or horizontary) consecutive cells with any width brush at a time.

NOTE:

  • Don't paint undesignated cells.
  • The cell may be painted multiple times.

Input: Designated cells (str).

Output: Minimum number of steps (int).
